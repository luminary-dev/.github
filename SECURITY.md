# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security problems.

- Preferred: use GitHub's private vulnerability reporting ("Report a vulnerability" under the Security tab of the affected repository).
- Or email **luminary2024@gmail.com** with details and steps to reproduce.

You will get an acknowledgement within 72 hours. Please give us a reasonable window to ship a fix before any public disclosure.

## Scope

All repositories in this organization, including the Baas.lk platform (`service-hub` and the `service-hub-*` service mirrors). Only the latest `main` of each repository is supported.

## Notes for Baas.lk service repos

The `service-hub-*` repositories are read-only mirrors of the [service-hub](https://github.com/luminary-dev/service-hub) monorepo — report issues against the affected service repo or the monorepo; fixes land in the monorepo and sync out.
